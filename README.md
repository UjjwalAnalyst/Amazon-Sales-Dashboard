# Amazon-Sales-Analysis
End-to-end sales data analysis project using Python, MySQL, and Power BI to analyze Amazon sales trends, customer behavior, and product performance.

## Project Overview

This project analyzes Amazon sales data to uncover insights about product performance, customer behavior, and sales trends.
The analysis was performed using SQL for data exploration and Power BI for building an interactive dashboard.

## Tools Used

* Python (Data loading and MySQL connection using SQLAlchemy)
* MySQL (Data analysis using SQL queries)
* Power BI (Data visualization and dashboard creation)

## Dataset

The dataset contains information about:

* Order ID
* Product Category
* Price and Discounts
* Quantity Sold
* Customer Region
* Payment Method
* Ratings and Reviews
* Total Revenue

## Key Business Questions

The analysis answers the following questions:

1. What is the total revenue generated?
2. How many orders were placed?
3. Which product categories generate the highest revenue?
4. Which regions contribute the most sales?
5. Which payment methods are most commonly used?
6. Do discounts influence the quantity sold?
7. What is the monthly revenue trend?

## Dashboard Metrics

The Power BI dashboard includes:

* Total Revenue
* Total Orders
* Total Units Sold
* Average Rating

## Visualizations

The dashboard includes:

* Monthly Revenue Trend
* Revenue by Product Category
* Revenue by Customer Region
* Payment Method Distribution
* Customer Rating by Product Category
* Discount vs Quantity Sold (Scatter Plot)
  
## Key Insights

* Some product categories generate significantly higher revenue than others.
* Sales distribution varies across different regions.
* Digital payment methods dominate customer transactions.
* Discounts show a relationship with quantity sold.

## Project Workflow

Dataset → Python → MySQL → SQL Analysis → Power BI Dashboard
